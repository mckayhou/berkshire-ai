# reports/ — 研究报告输出目录

各 Skill 生成的研究报告写入此目录。命名与目录约定见 [`../docs/report-conventions.md`](../docs/report-conventions.md)。

> 本 fork **未并入 upstream 的 2084 份历史报告和个人实盘记录**（那些属于原作者的研究成果/私人数据）。
> 这里只保留 **2 份完整样例报告**作为格式参考，其余报告由你自己跑 Skill 产出。

## 样例（格式参考）

- `RocketLab/RKLB-investment-research.md` — 综合深度研究样例（来自 upstream）
- `赛力斯/赛力斯-research-sample.md` — 综合深度研究样例（来自 upstream）

## 目录约定速览

- 公司报告：`reports/{公司名}/{公司名}-{类型}-{YYYYMMDD}.md`
- 行业/主题/组合报告：放 `reports/` 根目录
- 完整规范见 `docs/report-conventions.md`
